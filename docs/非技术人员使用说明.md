# 考勤统计助手使用说明

这份说明是给日常使用软件的同事看的，不需要懂代码。

你只需要做 3 件事：

1. 先确认当前年假表是否需要更新
2. 选择年份和月份，并上传这个月的 2 个 Excel 文件
3. 点击按钮生成结果

## 一、这个工具是做什么的

这个工具用来自动生成：

- 每个月的考勤明细
- 每个月的月度统计
- 整个年度的累计统计

生成后的结果文件固定为：

- `考勤统计结果.xlsx`

## 二、每个月要准备什么文件

每个月都需要准备下面 2 个文件：

1. `考勤表`
2. `请假表`

文件名不要求完全固定，但建议文件名里带这些关键字：

- `考勤`
- `请假`

此外，还有 1 个单独维护的“当前年假表”：

3. `员工年假总数表.xlsx`

注意：

- 月度文件每个月只要 2 个
- 当前年假表不是每个月都要重新传
- 只有员工年假信息发生变化时，才需要更新当前年假表
- 同类文件不要放两个
- 当前年假表文件名建议包含 `年假`

## 三、怎么打开程序

### Windows

优先双击：

1. `考勤统计助手.exe`

如果还没有 exe，再双击：

2. `运行考勤统计.bat`

## 四、打开程序后怎么操作

### 第1步：选择年份和月份

例如：

- 年份选 `2026`
- 月份选 `03`

选择完成后，界面会直接显示：

- 当前月份总体状态
- `考勤文件` 是否已找到
- `请假文件` 是否已找到

### 第2步：先确认当前年假表

如果界面里已经显示当前年假表信息，而且年假数据没有变化，就不用重新上传。

如果年假数据发生了变化，点击：

- `上传当前年假表`

### 第3步：上传这个月的 2 个文件

点击：

- `上传所选月份2个表`

然后按顺序选择：

1. `考勤打卡记录表`
2. `请假记录表`

如果你不想通过弹窗选择，也可以点击：

- `打开所选月份文件夹`

然后把 2 个文件放进去。

### 第4步：先检查当前年份文件

点击：

- `检查当前年份文件`

程序会直接告诉你：

- 当前这一年已经识别到哪些月份
- 哪些月份还需要处理

如果月份很多，还可以勾选：

- `仅显示待处理月份`

这时月份列表里只显示有问题的月份。

### 第5步：生成结果文件

点击：

- `生成结果文件`

程序会自动检查：

- 当前年假表是否已存在
- 当前选中年份下，哪些月份已经完整可统计
- 哪些月份还有缺少文件
- 哪些月份有重复文件

如果有问题，程序会直接弹出提示。

说明：

- 当前选中年份里，已经准备好的月份会先进入统计
- 还没准备好的月份不会丢失，会继续保留在待处理列表里

### 第6步：查看结果

生成完成后，程序会弹出结果窗口，显示：

- 结果文件位置
- 本次统计了哪些月份
- 哪些月份仍待处理

也可以点击：

- `打开生成好的 Excel`

## 五、结果文件里会有什么

结果文件里通常会有：

1. `2026合计`
2. `2026-02考勤明细`
3. `2026-02月度统计`
4. `2026-03考勤明细`
5. `2026-03月度统计`

说明：

- `2026合计` 是全年累计结果
- 每个月都会有 `考勤明细` 和 `月度统计`
- 新增一个月份后，再生成一次，全年合计会自动更新

## 六、以后每个月怎么做

通常每个月重复下面 3 步即可：

1. 选择该月的年份和月份
2. 点击 `上传所选月份2个表`
3. 点击 `检查当前年份文件`
4. 点击 `生成结果文件`

如果年假数据发生变化，再额外点击：

5. `上传当前年假表`

## 七、常见问题

### 1. 提示“还没有上传当前年假总数表”

请先点击：

- `上传当前年假表`

### 2. 提示“没有找到可统计的月份”

请检查：

- 这个月的 2 个文件是不是都已经上传成功
- 选择的年份和月份是不是正确
- 文件名里是不是没有 `考勤` 或 `请假` 这些关键字

### 3. 提示“缺少文件”或“重复文件”

请检查这个月份文件夹里是否：

- 缺少考勤文件
- 缺少请假文件
- 或放了两个同类文件

### 4. 结果里没有包含某个月

请检查：

- 你上传时选的年份和月份是不是正确
- 这个月的 2 个文件是不是都已上传成功
- 生成时程序有没有报错

### 5. 双击程序没有反应

Windows 电脑优先双击：

- `考勤统计助手.exe`

如果没有 exe，再双击：

- `运行考勤统计.bat`
